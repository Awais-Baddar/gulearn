# Gulearn

Gulearn is a mini online language learning and exchange platform where users connect with people worldwide to practice and learn new languages through real conversations.

The app focuses on user profiles, language-based matching, and social interaction in a clean and modern UI.

---

## Features
- User authentication (login & signup)
- Profile onboarding with avatar generation
- Native & learning language selection
- Discover language partners
- Send and manage friend requests
- Dashboard with instructors and connections
- Clean, responsive UI

---

## Tech Stack
- React & TypeScript (Frontend)
- Node.js & Express (Backend)
- MongoDB
- REST APIs
